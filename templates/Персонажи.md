# Персонажи

Обзор всех персонажей проекта.

## Главные герои

```dataview
TABLE WITHOUT ID 
  file.link AS "Персонаж",
  role AS "Роль",
  age AS "Возраст",
  occupation AS "Занятие"
FROM ""
WHERE type = "Персонаж" 
  AND contains(file.path, "{{projectName}}/Персонажи")
  AND contains(tags, "главный_герой")
  AND file.name != this.file.name
SORT file.name ASC
```
